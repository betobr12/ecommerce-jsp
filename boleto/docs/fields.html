Observações:
 - No campo Carteira, onde ocorre o valor "Indiferente", significa que o sistema não irá validar o conteúdo do campo, porém é preciso verificar
    com seu gerente qual é o valor correto para o seu caso.
 - Para Agência e Código do Cedente, o número de dígitos é sempre o número total que deverá ser passado para o sistema, por exemplo:
    4 dígitos (DV calculado): Você passará 4 dígitos para o sistema e o mesmo calculará o quinto dígito, caso tenha recebido do banco um
    número de 5 dígitos, deverá passar apenas os 4 primeiros.
 - Para os casos onde o DV deve ser informado, informar apenas números, caso exista algum caractere separador, ele será inserido pelo
    sistema.
 - Para o Nosso Número, o número de dígitos é o total que deverá ser passado para o sistema, será completado com zeros à esquerda caso seja
         menor. O dígito verficador, quando necessário, será calculado pelo sistema.
